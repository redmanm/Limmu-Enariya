Kaayyoo Ijaarama gamtaa barattoota Limmuu Innaariyaa!


1) Namoota baratee jijjiirama fidu, ofii fi biyya isaan gumaacha godhu oomosiisuu

2) Barattoonni keenya akka wal baranii fi Tokkummaa cimaa ta'e gidduu barattoota isheettii mirkaneessuu

3) Hawaasa ilaalcha gadi aanaa Dhimma barnootaaf qabu guutummaatti jijjiruu

4) Ilmaan ishee hundi akka barnootaaf of qopheessan taasisuu

5) Akkuma jalqaba Tokkummaa fi qindoomina seenaa abbootii keenyaa irraa baranne, ammas Tokkummaa fi qindoomina biyyattii keessatti labsuu

6) Erga bara dheeraatii ukkaamfamtee jiraachuun jiraachuu ishee beekamaadha, kanaaf ol baatee akka muldhattuu fi kutaa itoopiyaa keessaa ishee hangafaa fi kan dagatamte ta'uu qaama Dhimmi ilaallatu hubachiisuun ilaalcha ishee jijjiiruu!

7) Gaaffii Abbootii fi hadhaliin bara meeqa gaafachaa turan, karaa salphaa ta'een gaafannee jijjiiruu. fakkeenyaaf Bu'uuraalee misoomaa kanneen akka ; Tajaajila Ibsaa, Daandii istaandaardii isaa eeggate, Tajaajila niitwoorkii, Buufatalee/dhaabbilee fayyaa fi Hospitaala, manneen barnootaa istaandaardii isaanii eeggatan, Tajaajila bishaanii fi kanneen biroo magaalaa hanga baadiyyaatti akka furamu taasisuu.

8) Erga caasaan duraa yeroof hundeeffamee booda, magaalaa Atinaagootti Waltajjiin bal'aan akka qopheeffamuuf karaa Biiroo barnootaa fi kantiibaa magaalaa waliin ta'uun erga mijeeffamee booda karaa "channel telegram" keenyaa kan nuuf ifoomudha!

9) Erga caasaan duraa fi waltajjiin adeemsifamee gamtaan barattoota limmuu innaariyaa ijaan wal argee booda "channel telegram" keenya namoota kommunikeeshiinii ta'anii qofa akka eyyamamu ni taasifama.

10) Gamtaan barattoota limmuu innaariyaa karaa ifaa fi bilisa ta'een eyyama mataa isaa kan fudhatuu fi chaappaa mataa isaa kan qabaatu ni taasifama! Karaa hoggantoota aanaa 5 tiin ( Limmuu kossaa, limmuu saqqaa, cooraa Botor , noonnoo Beenjaa fi Botor xoollay waliin mariin cimaan erga taasifamee booda gara murtee fi hojiitti kan seenamu ta'a.

11) Bakkeewwan Dargaggoonnii fi Humni nama hojjechuu danda'uu qisaasa'aa jirutti jechuunis namoonni yeroo akkaan xiqqaa hojii isaaniitii kennanii yeroo baay'ee qaban hojiin alatti dabarsaniif caasaan barattoota limmuu innaariyaa bira ga'uun akka gadi fageenyaan mara'achiufamuu fi yeroon al tokko mana darbe deebi'ee dhufuu akka hin dandeenye hubachiisuun akka yeroo isaaniitti sirnaan fayyadaman gochuu.

12) Akka Hawaasa keenyaatti namoonni baay'een isaanii kaayyoo mataa isaa akka hin qabne beekamaadha! Kanumaan wal qabatee gamtaan barattoota limmuu innaariyaa dargaggoonnii fi hawaasni isaa hundi kaayyoo mataa isaanii baafatanii akka sochoo'an taasisuuf kan hundeeffamedha!



13) Barattoonni sadarkaa barnoota olaanoo Yuunivarsiitii fi koolleejjii kan mootummaas ta'ee dhuunfaa, Barattoota sadarkaa lammaffaa fi sadarkaa tokooffaa irra jiran Hawaasa isaaniif adda duree akka ta'aniif gorsi garaa garaa addatti ni kennama. Kana jechuun barattoonni kun haalli jiruu fi jireenya isaanii , akkaataan dubbii, waliin jireenya hawaasummaa, akkaataa uffannee fi rifeensa sirreeffannaa, safuu fi safeeffannaa, ilaalcha gaarii , amanamummaa, nageenya biyyattiif wabii , Tokkummaa fi jaalala akkasumas wal- danda'uu f k f n kan hawaasni naannoo fi namoonni biroon irraa baratan akka ta'uuf xiyyeeffannoo addaatiin ni hojjetama.

14) Barattoota maatii haraka qalleeyyii irraa barachuuf haalli hin mijanneef karaa koree aanaalee shanan irraa adda baasuun akka gargaaraman ni godha.

15) Barattoonni Yuunivarsiitii fi koolleejjii baratan yeroo boqonnaaf gana maatii galan bakka jiranitti barattoota qindeessuun akka hubannaa barnootaa akka kennan ni godhama.



16 ) Gamtaan barattoota limmuu innaariyaa Siyaasa, Amataa , Dhimmoota dhuunfaa, Ilaalcha fi ofittummaa f k k f irraa bilisa ta'ee kan hundeeffamedha.

Mul'ata

Gamtaan barattoota limmuu innaariyaa Hawaasa Dinagdeen, siyaasaan, barnootaa fi haala jiruu fi jireenya mijataa keessa akka jiraatuu fi ilmaan isaanii dukkana wallaalummaa keessaa haala kamiin ba'uu akka danda'anii fi Qaamni baratu/ttu kamuu kaayyoo mataa isaa akka qabaatu taasisuu. Barattoonni sadarkaa kamirrayyuu jiranu biyya isaaniif itti gaafatamummaa guddaa akka qabanuu fi moodela hawaasaa akka ta'an gochuu irratti kan xiyyeeffatedha.

Bekumsa

Barattoota dandeettii fi gahumsa qaban oomishuudha. Barattoonni keenya sadarkaa 1ffaa hanga eebbifamanii bahanii hojii eegalanitti Gamtaan kun duuka bu'uun isaan akka deeggaruuf akkasuma Bulchiisa aanaa hunda waliin dubbatee barattonni seektara seectaraan ogummaa qabaniin akka hojii eegalan hoji dhabdummaan akka hin jiraanne gochuun kaayyoo isa hangafa dha.

Deggarsa

Hundaafuu yaadaf qalbiin qaamaanis yoo barbaachise inumaa horiinis ni deeggarra gamataa keenya kan guddisu numa qofa. Generationy dhufuuf wanni ol keenyu hin jiru Abbootiin nuuf ol ka'an yoo ta'e nus dabarsinee ol keeny Limmuu Inaariyaa kutaa qaamaa biyyatti keessaa isii hanagafa akkaaumas karaa maraan kan guddattee fi dagaaginni barnootaa sirrumatti lafa kan qabate, kan ummanni isaa nama baratee fi ummata addunyaa kanaaf seenaa bu'aa qabu hojjetee darbuu keessatti oomishuudha.
